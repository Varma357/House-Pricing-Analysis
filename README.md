# Bengaluru House Price Prediction | Data Analytics & Machine Learning Project

## Project Overview
This project analyzes Bengaluru real estate data and builds a machine learning model to predict house prices based on property features such as location, size (BHK), square footage, and number of bathrooms. The project demonstrates end-to-end data analytics, feature engineering, and predictive modeling using Python.

---

## Business Problem
Property prices in Bengaluru vary significantly across locations and property types. Buyers, sellers, and investors often struggle to estimate fair property values.

### Objectives:
- Identify key factors influencing house prices
- Predict house prices using historical real estate data
- Provide data-driven insights for real estate decision-making

- Location has the highest impact on property pricing
- Price per square foot varies significantly across neighborhoods
- Property size influences price but not always linearly
